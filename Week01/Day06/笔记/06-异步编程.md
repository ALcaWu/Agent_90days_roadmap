## 一、为什么需要异步？

### 同步 vs 异步

| 模式 | 特点 | 场景 |
|------|------|------|
| 同步 | 顺序执行，阻塞等待 | CPU密集型 |
| 异步 | 并发执行，不阻塞 | I/O密集型（API调用、文件读写） |

**AI开发典型场景：**
- 批量调用多个LLM API（同步要排队，异步可并发）
- 同时读取多个文档
- 并发处理多个用户请求

## 二、asyncio 核心概念

### 2.1 async/await 语法

```python
import asyncio

# 定义异步函数
async def fetch_data():
    await asyncio.sleep(1)  # 模拟I/O操作
    return "data"

# 运行异步函数
result = asyncio.run(fetch_data())
```

### 2.2 并发执行多个任务

```python
import asyncio
import aiohttp

async def fetch_url(session, url):
    async with session.get(url) as resp:
        return await resp.text()

async def main():
    async with aiohttp.ClientSession() as session:
        urls = ["https://api.example.com/1", "https://api.example.com/2"]
        # 并发执行
        results = await asyncio.gather(*[fetch_url(session, u) for u in urls])
        return results

asyncio.run(main())
```

### 2.3 Task 控制

```python
async def main():
    task = asyncio.create_task(some_async_func())
    # ... 做其他事
    result = await task  # 等待结果
```

## 三、asyncio 常见API

| API | 作用 |
|-----|------|
| `asyncio.run()` | 运行异步主函数 |
| `asyncio.sleep()` | 异步sleep（不阻塞） |
| `asyncio.gather()` | 并发执行多个协程 |
| `asyncio.create_task()` | 创建后台任务 |
| `asyncio.wait()` | 等待多个任务完成 |

## 四、同步代码转异步

### 4.1 requests → aiohttp

```python
# 同步版
def sync_fetch(urls):
    results = []
    for url in urls:
        r = requests.get(url)
        results.append(r.json())
    return results

# 异步版
async def async_fetch(session, url):
    async with session.get(url) as resp:
        return await resp.json()

async def async_fetch_all(urls):
    async with aiohttp.ClientSession() as session:
        tasks = [async_fetch(session, u) for u in urls]
        return await asyncio.gather(*tasks)
```

### 4.2 time.sleep → asyncio.sleep

```python
# 同步
time.sleep(1)

# 异步
await asyncio.sleep(1)
```

## 五、异步LLM调用实战

```python
import asyncio
import aiohttp
import os

API_KEY = os.getenv("OPENAI_API_KEY")
URL = "https://api.openai.com/v1/chat/completions"

async def call_llm(session, messages):
    headers = {"Authorization": f"Bearer {API_KEY}"}
    payload = {"model": "gpt-4o-mini", "messages": messages}
    
    async with session.post(URL, json=payload, headers=headers) as resp:
        return await resp.json()

async def batch_call_llm(prompt_list):
    async with aiohttp.ClientSession() as session:
        tasks = [
            call_llm(session, [{"role": "user", "content": p}])
            for p in prompt_list
        ]
        return await asyncio.gather(*tasks)

# 批量生成5个回答
prompts = ["解释量子计算"] * 5
results = asyncio.run(batch_call_llm(prompts))
print(len(results))  # 5
```
